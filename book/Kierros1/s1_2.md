# 1.2 Muuttujat
Ohjelmoidessa tallennamme tietoa *muuttujiin* (engl. *variable*). Esimerkiksi *input*-funktio tallentaa tﺣ۳ssﺣ۳ esimerkissﺣ۳
kﺣ۳yttﺣ۳jﺣ۳n syﺣﭘtteen merkkijonona muuttujaan, jonka nimi on etunimi:
``` ipython3
etunimi = input("Anna etunimesi\n")
```
Tavallisia muuttujatyyppejﺣ۳ Pythonissa ovat:

| Muuttujatyyppi | Nimi Pythonissa | Esimerkkejﺣ۳ | Kommentti |
| :---: | :---: | :---: | :---: |
| Merkkijono | **str** | "Hei!", 'OK' | Sekﺣ۳ "kaksinkertaiset" ettﺣ۳ 'yksinkertaiset' lainausmerkit toimivat |
| Kokonaisluku | **int** | 2, 0, -2, 1924	| Positiiviset ja negatiiviset kokonaisluvut ja nolla |
| Liukuluku | **float** | 1.04, -3.0004	| Desimaaliluku |
| Kompleksiluku | **complex** | 2.0 + 3.0j | Emme kﺣ۳ytﺣ۳ tﺣ۳llﺣ۳ kurssilla |
| Totuusarvo | **bool** | True tai False | Englannin kielen sanasta Boolean |

Muuttujiin voidaan sijoittaa arvoja "="-merkin avulla. Muutama esimerkki muuttujien kﺣ۳ytﺣﭘstﺣ۳:
```{code-cell} ipython3
:tags: ["auto-execute-page"]
iso_luku = 50000005
print("Iso lukumme on", iso_luku)
pieni_luku = 0.0009
print("Pieni lukumme on", pieni_luku) 
```
Muuttuja **iso_luku** on yllﺣ۳olevassa kokonaisluku, kun taas muuttuja **pieni_luku** on liukuluku.
:::{admonition} Huom!
:class: note
Toisin kuin monissa muissa ohjelmointikielissﺣ۳, Pythonissa muuttujan tyyppiﺣ۳ ei tarvitse mﺣ۳ﺣ۳ritellﺣ۳ ennen muuttujan
kﺣ۳yttﺣ۳mistﺣ۳. Python pﺣ۳ﺣ۳ttelee muuttujan tyypin, kun muuttujan arvo asetetaan.
:::

## Lukuarvojen yksikkﺣﭘjen huomioiminen
Tieteellisessﺣ۳ laskennassa meidﺣ۳n pitﺣ۳ﺣ۳ aina olla selvillﺣ۳ kﺣ۳ytﺣﭘssﺣ۳ olevista yksikﺣﭘistﺣ۳. Python ei pidﺣ۳ mitenkﺣ۳ﺣ۳n kirjaa
muuttujan yksikﺣﭘistﺣ۳, vaan se on ohjelmoijan vastuulla. Onkin erittﺣ۳in tﺣ۳rkeﺣ۳ﺣ۳ kirjata yksikﺣﭘt aina muistiin kommentteina. 
Esimerkiksi:
``` ipython3
# Tehtﺣ۳vﺣ۳n lﺣ۳htﺣﭘarvot
n = 0.334  # ainemﺣ۳ﺣ۳rﺣ۳, mol
V = 0.014  # tilavuus, m^3
p = 101325 # paine, Pa
```
On erittﺣ۳in suositeltavaa pitﺣ۳ﺣ۳ kaikki lukuarvot **SI-yksikﺣﭘissﺣ۳** aina kun mahdollista.
