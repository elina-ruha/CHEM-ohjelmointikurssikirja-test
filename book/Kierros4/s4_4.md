# 4.4 Laskuoperaatiot NumPy-taulukoilla
Luodaan ensin uusi taulukko (yksiulotteinen vektori). 
```{code-cell} ipython3
:tags: ["auto-execute-page"]
import numpy as np

v1 = np.arange(100, 1001, 100)
print("v1 on", v1)
```
NumPy-taulukoiden ja yksittäisten lukuarvojen laskuoperaatiot onnistuvat suoraviivaisesti. 
NumPy suorittaa laskuoperaation jokaiselle alkiolle:
```{code-cell} ipython3
v2 = v1 + 1
print("v2 on", v2)
v3 = v1 * 2
print("v3 on", v3)
v4 = v1 / 100
print("v4 on", v4)
v42 = 100 / v1
print("v42 on", v42)
```
NumPy-taulukoita voi myos lisätä, kertoa ja jakaa keskenään. Operaatiot tehdaan alkioittain, joten taulukoiden tulee olla samankokoisia!
```{code-cell} ipython3
v5 = v2 + v2
print("v5 on", v5)
v6 = v4 * v4
print("v6 on", v6)
v7 = v3 / v1
print("v7 on", v7)
```
