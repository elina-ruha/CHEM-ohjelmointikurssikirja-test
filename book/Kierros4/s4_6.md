# 4.6 Matplotlib-kirjasto
Matplotlib-kirjasto sisältää erittäin monipuoliset työkalut erilaisten kuvaajien tekemiseen. Tällä kurssilla tutustumme
*matplotlib.pyplot*-moduuliin, joka mahdollistaa kuvaajien piirtämisen hieman Matlabin tapaan. 

Kun teet Matplotlib-tehtäviä Spyderissä, kuvaajien pitäisi aueta siististi suoraan IPython-konsolin yllä olevalle Plots-välilehdelle. Asiasta kerrotaan lisää [Spyderin käyttöohjeissa](../Lisamateriaali/spyder_käyttö.md). 

Jos ajat esimerkin
```
import numpy as np
import matplotlib.pyplot as plt

X = np.arange(1, 101)
Y = np.sqrt(X)
plt.plot(X, Y)
plt.show()
```
konsoli tulostaa:
```
Figures now render in the Plots pane by default. 
To make them also appear inline in the Console, 
uncheck "Mute Inline Plotting" under the Plots pane options menu.
```
ja kun klikkaat **Plots-välilehteä** konsolin yläpuolelta, näet siellä kuvaajan:

```{figure} ../figures/4_6kuva.png
---
width: 100%
align: center
---
```
:::{admonition} Huom!
:class: note
Jos haluat, että matplotlib piirtää kuvaajat suoraan IPython-konsoliin, seuraa Spyderin antamia ohjeita

Jos et käytä Spyderiä vaan jotain muuta kehitysympäristöä (esim. PyCharm tai VS Code), sinun pitää itse selvittää, miten
Matplotlib-kuvaajat toimivat sen kanssa. Voi olla, että ne avautuvat omina ikkunoinaan.
:::
