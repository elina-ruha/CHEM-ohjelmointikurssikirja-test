# Vaihtoehtoja Spyderille
On henkilöstä riippuvaa, mikä kehitysympäristö tuntuu miellyttävältä käyttää. Tämän takia esittelemme myös vaihtoehtoja Spyder-kehitysympäristölle.
Jos olet tarvinnut koodausympäristöä aikaisemmin, voi olla, että olet jo asentanut jonkun seuraavista vaihtoehdoista. Tällöin voit aivan hyvin käyttää
sitä tehtävien ratkaisemiseen, kunhan löydät itse sieltä tarvittavat välineet kuten esimerkiksi kuvaajien Plots-välilehden.

## PyCharm
- Spyderin kanssa samantasoinen kehitysympäristö, jossa monipuolisia, visuaalisia ominaisuuksia
- Koodin virheenjäljitys (debugger), Python-konsoli, tieteellisessä tilassa datan visualisointi ja muuttujat-välilehti
- Ilmainen (maksullisessa versiossa lisää ominaisuuksia, joita ei kurssilla tarvita)
- Asennus: <https://www.jetbrains.com/pycharm/download/?section=windows>
- Lisätietoa: <https://www.jetbrains.com/pycharm/>
## Visual Studio Code (VS Code)
- Microsoftin koodin kehitysympäristö, johon kannattaa ladata Python-laajennus tehtäviä varten
- Debugger, Python-konsoli, muuttujien hallinta, monet toiminnot komentojen kautta
- Ilmainen, visuaalinen käyttö
- Asennus: <https://code.visualstudio.com/Download>
- Lisätietoa: <https://code.visualstudio.com/>
